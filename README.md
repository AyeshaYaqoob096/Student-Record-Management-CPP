# Student Record Management System

## 📌 Short Description
This C++ application introduces foundational concepts of data grouping using structured data types (`struct`). The project demonstrates how to define a custom composite data type, instantiate it, perform compile-time member initialization, and retrieve data points using the dot (`.`) member access operator.

---

## 🛠️ Features
* **Custom Data Structuring:** Utilizes a `struct` to unify multiple disparate data types (integers and character arrays) into a single cohesive entity.
* **Static Initialization:** Demonstrates sequential attribute population during object instantiation.
* **Console Reporting:** Accesses and formats structured data cleanly for terminal readouts.
